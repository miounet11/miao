# 日本語言語パック

Miaoda IDE の日本語言語パックです。

## 使用方法

この拡張機能をインストールした後、`Ctrl+Shift+P`（Windows/Linux）または `Cmd+Shift+P`（macOS）でコマンド パレットを開き、「Configure Display Language」と入力して「日本語」を選択し、Miaoda IDE を再起動してください。

## 機能

- Miaoda IDE のコア インターフェイスを翻訳
- メニュー バー、コマンド パレットを翻訳
- サイド バーとステータス バーを翻訳
- 一般的なコマンドと操作を翻訳

## フィードバック

翻訳の問題や改善の提案がある場合は、[GitHub](https://github.com/miaoda/miaoda-ide) で Issue を作成してください。
