# 中文（简体）语言包

Miaoda IDE 的中文简体语言包。

## 使用方法

安装此扩展后，按 `Ctrl+Shift+P`（Windows/Linux）或 `Cmd+Shift+P`（macOS）打开命令面板，输入 "Configure Display Language" 选择 "中文（简体）"，然后重启 Miaoda IDE。

## 功能

- 翻译 Miaoda IDE 核心界面
- 翻译菜单栏、命令面板
- 翻译侧边栏和状态栏
- 翻译常用命令和操作
