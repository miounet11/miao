# English Language Pack

English Language Pack for Miaoda IDE.

## Usage

This is the default language pack for Miaoda IDE. English is used by default.

To switch to another language, press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS) to open the Command Palette, type "Configure Display Language", select your preferred language, and restart Miaoda IDE.

## Features

- Default English interface for Miaoda IDE
- Provides base translations for all UI elements
- Menu bar, command palette, and status bar in English
- Standard commands and operations in English
