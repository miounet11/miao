# Miaoda IDE

AI-powered development environment with multi-agent orchestration.

## Features

- **Multi-Agent Orchestration**: Coordinate multiple AI agents for complex tasks
- **Smart Skills System**: Generate and manage reusable AI skills
- **Context-Aware**: Intelligent code understanding and suggestions
- **Cost Optimization**: Transparent AI usage tracking and optimization
- **Quality Guardian**: Automated code quality checks

## Quick Start

1. Install the extension
2. Press `Cmd+Shift+Q` (Mac) or `Ctrl+Shift+Q` (Windows/Linux) for quick actions
3. Start with "Miaoda: Show Quick Actions" command

## Requirements

- VSCode 1.85.0 or higher
- API keys for AI providers (OpenAI, Anthropic, etc.)

## Configuration

Set your API keys in Settings:
- `miaoda.apiKeys.openai`
- `miaoda.apiKeys.anthropic`
