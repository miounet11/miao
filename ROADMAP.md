# Miaoda IDE 版本路线图 / Roadmap

> 基于 v1.0.0 的迭代优化和升级计划

---

## v1.1.0 — 稳定性与体验优化 (2026 Q2)

- [ ] 用户登录/注册系统完善（邮箱验证、OAuth）
- [ ] 云端存储配额管理与用量提醒
- [ ] Dashboard UI 优化，增加图表可视化
- [ ] 离线模式增强，断网时自动切换本地存储
- [ ] 多语言 UI 完善（中文/英文/日文）
- [ ] 自动更新机制 (auto-update)

## v1.2.0 — 协作与团队功能 (2026 Q3)

- [ ] 团队工作区共享
- [ ] 实时协作编辑 (CRDT)
- [ ] 项目模板市场
- [ ] 团队成本分摊与统计
- [ ] 权限管理 (RBAC)

## v1.3.0 — AI 能力增强 (2026 Q4)

- [ ] 本地模型运行时 (Llama, CodeLlama, DeepSeek Coder)
- [ ] 真实 Embedding 模型集成 (CodeBERT, UniXcoder)
- [ ] AI 代码审查自动化
- [ ] 智能重构建议
- [ ] 多模型 A/B 测试框架

## v2.0.0 — 下一代平台 (2027)

- [ ] 插件市场 (Miaoda Extensions Marketplace)
- [ ] Web 版本 (浏览器内运行)
- [ ] 移动端配套 App
- [ ] 企业版 (私有部署、SSO、审计日志)
- [ ] AI Agent 自定义工作流编排

---

## 长期愿景 / Long-term Vision

Miaoda IDE 致力于成为最灵活、最开放的 AI 编程工具：
- 零厂商锁定，支持任意 LLM
- 本地优先，云端增强
- 中国开发者友好，全球化视野
- 开源社区驱动
