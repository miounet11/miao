# Miaoda IDE Branding Guide

## Product Identity

**Product Name**: Miaoda (妙搭)
**English**: Miaoda IDE
**Chinese**: 妙搭 IDE
**Developer**: Coco (来自中国)
**Tagline**: "Universal LLM Integration - Your Way"
**Chinese Tagline**: "通用 LLM 集成 - 随心所欲"

## Brand Story

Miaoda (妙搭) means "wonderful assembly" or "clever combination" in Chinese. It represents the IDE's core philosophy: seamlessly integrating ANY language model with the correct API key, URL, and model name. No restrictions, no limitations - just pure flexibility.

Developed by Coco from China, Miaoda IDE empowers developers worldwide to choose their preferred AI models without vendor lock-in.
