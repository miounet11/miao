## Upgrade Guide

### From 0.8.x to 0.9.0

**Breaking Changes**: None

**New Features**:
1. Hybrid Model Architecture - Configure in settings
2. Code Knowledge Graph - Run "Build Knowledge Graph"

**Improvements**:
- Context engine is 2.2x faster
- Auto-fix covers 12% more issues

**Migration Steps**:
1. Update extension
2. Restart VSCode
3. No configuration changes needed

### From 0.5.x to 0.8.0

**Breaking Changes**: None

**New Features**:
1. Smart Context Engine
2. Cost Optimization
3. Quality Guardian
4. Progressive Onboarding

**Migration Steps**:
1. Update extension
2. Complete onboarding (optional)
3. Configure API keys if needed
